# Capturas de pantalla

Imágenes de la aplic durante pruebas manuales (búsqueda de animales, gato del día, películas, etc.). Los nombres siguen el patrón `Screenshot_YYYYMMDD_HHMMSS.png`.

Para añadir una captura nueva, guarda el PNG en esta carpeta y haz commit junto con el cambio de código o documentación relacionado.
